# Spring E-Commerce Backend (Microservices)

This repository contains a **Spring Boot 4.x based E-Commerce Backend System** built using **Microservices Architecture**.

The primary goal of this project is **learning and mastering modern Spring Boot microservices**, while also serving as a **reference project for interviews and real-world system design**.

---

## 🎯 Objective

- Learn and master **Spring Boot 4.x** with **Java 25**
- Design and implement a **real-world microservices architecture**
- Understand **service discovery, API gateway, security, resilience, and scalability**
- Prepare confidently for **Java / Spring Boot / Microservices interview questions**
- Provide a **free, open-source learning project** for anyone interested

---

## 🧱 Architecture Overview

This project follows a **microservices-based architecture** with:

- Independent services
- Centralized API Gateway
- Service Discovery
- Distributed databases
- Docker-based local development


---

## 🧩 Planned / Implemented Microservices

| Service Name | Responsibility |
|-------------|----------------|
| Service Discovery | Service registration & discovery |
| API Gateway | Routing, authentication, rate limiting |
| Auth Service | Authentication & authorization |
| User Service | User & profile management |
| Product Service | Product catalog |
| Order Service | Order lifecycle management |
| Inventory Service | Stock management |
| Payment Service | Payment processing |
| Notification Service | Email / event notifications |

*(Services will be added and evolved incrementally)*

## 🚀 Getting Started (High Level)

1. Clone the repository
2. Start infrastructure services (Docker)
3. Start Service Discovery
4. Start API Gateway
5. Start individual microservices

Each service is **independently runnable**.
